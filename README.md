# ConsoleChatGPT
ChatGPT simulator using OpenAI Official API model: text-davinci-003, but generated with proper prompt.

## Get your api key
visit https://platform.openai.com/account/api-keys

## How to start
```bash
export OPENAI_API_KEY=YOUR_KEY
python3 src/chat.py
```
